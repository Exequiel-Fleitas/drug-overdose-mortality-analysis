# drug-overdose-mortality-analysis
Statistical analysis of U.S. drug overdose mortality trends by age and sex using R.

# Drug Overdose Mortality Analysis

This project investigates how age and sex influence drug overdose death rates in the United States from 1999–2018.

## Methods
- Data Cleaning and Wrangling
- Exploratory Data Analysis
- Multiple Linear Regression
- ANOVA
- Hypothesis Testing

## Tools
- R
- dplyr
- ggplot2

## Key Findings
- Males had significantly higher overdose death rates than females.
- Adults aged 35–64 experienced the highest overdose mortality rates.
- Age and sex were significant predictors of overdose deaths.
